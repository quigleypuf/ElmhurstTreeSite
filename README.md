# ElmhurstTreeSite
This is a basic website where you can search for different trees around Elmhurst's campus. 755/755 Trees have been successfully scraped and added to the database.

Site is being hosted on Amazon AWS EC2 instance @ http://3.16.128.43/

Use locally:
1. Download the files
2. Download python, and SQLite
3. Open terminal
4. Navigate to the project folder
5. Type: "python3 -m http.server 8000
6. Open localhost:8000 on a webbrowser
